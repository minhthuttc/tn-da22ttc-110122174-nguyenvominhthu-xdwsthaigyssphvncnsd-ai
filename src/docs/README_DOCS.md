## 📝 Checklist trước khi nộp:

### File .docx:
- [ ] Trang bìa đầy đủ thông tin
- [ ] Mục lục tự động (Insert → Table of Contents)
- [ ] Đánh số trang
- [ ] Danh mục hình, bảng đầy đủ
- [ ] Font chữ Times New Roman 13pt
- [ ] Lề: Trái 3cm, Phải 2cm, Trên 2cm, Dưới 2cm
- [ ] Kiểm tra chính tả (F7)
- [ ] Tài liệu tham khảo đúng format APA/IEEE

### File .pdf:
- [ ] Fonts nhúng đầy đủ
- [ ] Hình ảnh không bị mờ
- [ ] Bookmarks/mục lục tự động
- [ ] Kích thước file dưới 10MB

### File .pptx:
- [ ] 15-20 slides
- [ ] Không quá nhiều text trên 1 slide
- [ ] Hình ảnh chất lượng cao
- [ ] Transition mượt mà
- [ ] Notes cho presenter (nếu cần)

### File Poster PDF:
- [ ] Kích thước đúng A1 (594x841mm)
- [ ] Resolution 300 DPI
- [ ] Màu sắc CMYK (cho in)
- [ ] Text đọc được từ xa 2-3 mét
- [ ] QR code test được

### File HUONG_DAN_SU_DUNG.md:
- [ ] Các bước cài đặt đầy đủ
- [ ] Screenshots minh họa
- [ ] Tài khoản demo
- [ ] Xử lý lỗi thường gặp
